# 🔍 FORENX - Digital Forensics Investigation Tool

![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)
![License](https://img.shields.io/badge/License-MIT-green.svg)
![Status](https://img.shields.io/badge/Status-Active-success)

FORENX is an advanced Digital Forensics Investigation Tool developed in Python. It is designed to assist cybersecurity students, researchers, and investigators in collecting, analyzing, and preserving digital evidence through an intuitive graphical interface.

---

# 📌 Features

## 🔎 File Analysis
- View detailed file metadata
- Calculate MD5, SHA1, and SHA256 hashes
- Detect duplicate files
- Analyze file properties

## 💾 Data Recovery
- Recover deleted files
- Scan storage devices
- Preview recoverable files
- Export recovered data

## 🧾 Hash Generator & Verifier
- Generate secure hashes
- Verify file integrity
- Compare multiple hashes

## 🖥️ System Information
- CPU Information
- RAM Usage
- Disk Information
- Operating System Details
- Network Information

## 🌐 Network Analysis
- IP Configuration
- Active Network Connections
- Running Services
- DNS Information

## 🔐 Evidence Collection
- Create forensic reports
- Export evidence logs
- Generate timestamps
- Preserve chain of custody

## 📄 Report Generation
- PDF Reports
- CSV Reports
- JSON Reports
- HTML Reports

## 🎨 Modern GUI
- Dark Theme
- User-Friendly Interface
- Real-Time Progress
- Responsive Design

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/YourUsername/FORENX.git
```

```bash
cd FORENX
```

---

## Create Virtual Environment

Windows

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

---

## Install Requirements

```bash
pip install -r requirements.txt
```

---

## Run

```bash
python main.py
```
